Usando o elemento <HEAD>

O elemento <head> é um container usado para guardar configurações gerais de um documento HTML. Dentro dele podemos usar pooutros elementos que configuram partes específicas além de configurar os metadados. Há alguns elemntos que podem ser usados fora desse container. Conheça a baixo alguns dos elementos mais comuns.

Exemplo Japonês:ご飯が熱い


Title

Define o título do documento, mostrado na barra de título de um navegador ou na aba da página. Pode conter somente texto e quaisquer marcações contidas no texto não são interpretadas.

O elemento <title> é sempre usado dentro da <head> da página.


Style

O elemento style pode ser usado fora do elemento HEAD. Esse elemento pode ser usado fora da área do <head>.


Script

O SCRIPT é usado para incorporar um script do lado do cliente. Esse elemento pode ser usado fora da área do <head>, assim como o <style>


Link

O elemento Link permite que sejam criados vínculos entre o documento atual e um recurso externo. Ele só pode ser usado pode dentro da área do <head> e podem existir várias ocorrências dele.


Meta

O elemento META define várias informações dos metadados. Esse elemento só pode ser usado dentro do <head>.
