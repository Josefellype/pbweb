Quando usamo o Grid Layout, podemos usar essa estrutura bidimensional a partir do CSS, mantendo a separação do conteudo e do layout.

Uma das vantagens de usar o Grid Layout é que ele nos ajuda a criar layouts que podem ser redefinidos usando Media Queries e se adaptar a diferentes contextos. Além disso, o Grid Layout nos permite organizar corretamente a disposição dos elementos que estarão visíveis na página. Isso deixa o Designer livre para alterar a localização dos elementos, fazendo com que os Breakpoints fiquem mais precisos.

Vale a pena destacar que o Grid Layout não tem o objetivo de substituir o flexbox.
