Layout Responsivo com o Flex Box

Primeiro passo

Para começar, precisamos selecionar quais elementos devem ser dispostos como containers flexíveis. Para fazer isso, definimos um valor especial de exibição no elemento pai, o container, dos elementos que você deseja afetar.

Segundo Passo

O container flexível vai ser estilizado. Assim, dentro do identificador que o representa, vamos inserir as regras CSS responsáveis por ajustar o visual dos elementos que ficarão dentro dele.

Terceiro Passo

Esse passo é dedicado em configurar os elementos que estão dentro do container flexível, isso é, seus filhos.

Apesar de todos terem o mesmo tratamento feito pelo Container Flexível, é possível fazer com que alguns deles tenham um visual que o diferencia dos demais.
